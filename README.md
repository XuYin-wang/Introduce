自己紹介Webページ

📌 概要

このプロジェクトは、HTML・CSS・JavaScriptを用いて作成した自己紹介用のWebページです。
シンプルな構成ながら、ページ切り替えやアニメーションを取り入れ、より実用的なUIを目指しました。

⸻

🚀 機能
	•	ページスライド切り替え（JavaScript）
	•	レスポンシブ対応（基本レベル）
	•	ボタンによる画面遷移
	•	シンプルで見やすいデザイン

⸻

🛠 使用技術
	•	HTML
	•	CSS
	•	JavaScript
